# KBS Strategy Tracking Dashboard

Dashboard ติดตามการส่งข้อมูลกลับ (แบบสำรวจสินค้าไม่เคลื่อนไหว) ของ **แผนกวิเคราะห์และกลยุทธ์ · KBS Sugar Group** ครอบคลุม 4 โรงงาน พร้อมฝัง Dead Stock Management Dashboard ของโรงงาน KBS และ KPP ไว้ในไฟล์เดียว

🔗 **เปิดใช้งาน:** https://<username>.github.io/<repo>/

---

## คุณสมบัติ

**หน้าติดตามการส่งข้อมูล**
- ตารางติดตามงาน 6 คอลัมน์: โรงงาน · ฝ่าย · แผนก · ผู้รับผิดชอบ · สถานะ · Remark
- เพิ่ม / แก้ไข / ลบ รายการผ่านฟอร์ม (สถานะ: ส่งแล้ว / ยังไม่ส่งกลับ / รอดำเนินการ)
- นำเข้า Excel (`.xlsx`) — โหลดข้อมูลจากไฟล์มาแทนที่ (อ่านหัวตารางภาษาไทยอัตโนมัติ)
- อัปเดต Excel — ดาวน์โหลดข้อมูลปัจจุบันกลับเป็นไฟล์ `KBS_ติดตามงาน_YYYY-MM-DD.xlsx`
- ค้นหา + กรองตามสถานะหรือรายโรงงาน
- การ์ดสรุปรายโรงงาน + แถบ progress + สรุปประเด็นค้าง

**หน้า Dead Stock Dashboard (ฝังในตัว)**
- **KBS** — บริษัท น้ำตาลครบุรี จำกัด (มหาชน) รวมทุกแผนก
- **KPP** — บริษัท ผลิตไฟฟ้าครบุรี (WHKPP)
- ABC Analysis, Moving Analysis, มูลค่าคงค้างตามคลัง, สัดส่วนประเภทสินค้า พร้อมกราฟ Chart.js

## การใช้งาน

เปิด `index.html` ในเบราว์เซอร์ได้เลย — ทำงานแบบ **offline 100%** (ฝัง Chart.js, SheetJS, โลโก้ และ dashboard ทั้งหมดไว้ในไฟล์เดียว ไม่ต้องต่อ CDN เหมาะกับเครือข่ายภายในโรงงาน)

## ข้อมูล

- ข้อมูลที่เพิ่ม/แก้ไขในหน้าติดตามงานจะบันทึกใน **localStorage** ของเบราว์เซอร์แต่ละเครื่อง (ไม่ sync ข้ามเครื่อง)
- ต้องการแชร์ข้อมูลร่วมกันหลายคน — ให้ export เป็น Excel แล้วส่งต่อ หรือต่อ backend (Google Sheets / SharePoint) แยกเพิ่ม
